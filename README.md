# 🤖 AI Chatbot Using Simple Rules

A professional rule-based AI Chatbot built using Python. The chatbot responds to user queries using predefined responses and `if-else` statements. This project was developed as part of an Artificial Intelligence Internship.

---

## 📌 Features

- Interactive chatbot interface
- Rule-based conversation
- Uses Python `if-else` statements
- Answers AI-related questions
- Provides current date and time
- Simple and beginner-friendly
- Clean modular code structure

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/YourUsername/AI-Chatbot-Using-Simple-Rules.git
```

2. Navigate to the project folder

```bash
cd AI-Chatbot-Using-Simple-Rules
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the chatbot

```bash
python chatbot.py
```
